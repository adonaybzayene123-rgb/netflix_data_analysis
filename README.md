### My First Netflix Data Project! 🎬

I just finished my first real data project using Python! I wanted to see what is actually going on behind the scenes of the Netflix library, so I used Pandas to explore the data and find some interesting trends.

#### 📊 What I discovered

* **Huge growth**: It was crazy to see how much more content Netflix started adding between 2016 and 2019.
* **Movies vs Shows**: I always thought there were more TV shows, but actually about 70% of the library is Movies.
* **Big players**: The US and India are the biggest producers of content on Netflix.
* **Messy data**: Real data is messy! I had to spend time fixing missing values for things like directors, actors, and countries before making any charts.

#### 🛠️ Tools I used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook
